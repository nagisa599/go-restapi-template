## エラー設計

| Error Type            | Code | Message               |
| --------------------- | ---- | --------------------- |
| Bad Request           | 400  | Bad request           |
| Unauthorized          | 401  | Unauthorized          |
| Forbidden             | 403  | Forbidden             |
| Not Found             | 404  | Not found             |
| Internal Server Error | 500  | Internal server error |
